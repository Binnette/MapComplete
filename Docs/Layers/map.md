### map\_type

The question is _What type of map is shown?_

*   _Topographical map
    
    The map contains contour lines.
    
    _corresponds with [map\_type](https://wiki.openstreetmap.org/wiki/Key:map_type)\=[topo](https://wiki.openstreetmap.org/wiki/Tag:map_type%3Dtopo)
*   _A map with all streets or ways of an area.
    
    The streets are mostly named; the angles, distances etc. are accurate
    
    _corresponds with [map\_type](https://wiki.openstreetmap.org/wiki/Key:map_type)\=[street](https://wiki.openstreetmap.org/wiki/Tag:map_type%3Dstreet)
*   \*This is a schematic map.
    
    A sketched map with only important ways and POIs. The angles, distances etc. are merely illustrative, not accurate.
    
    \* corresponds with [map\_type](https://wiki.openstreetmap.org/wiki/Key:map_type)\=[scheme](https://wiki.openstreetmap.org/wiki/Tag:map_type%3Dscheme)
This is a toposcope.

A marker erected on high places which indicates the direction to notable landscape features which can be seen from that point

corresponds with [map\_type](https://wiki.openstreetmap.org/wiki/Key:map_type)\=[toposcope](https://wiki.openstreetmap.org/wiki/Tag:map_type%3Dtoposcope)
